Our Work

IterātED: A Complete Iterative Assessment Platform

Stop grading static worksheets. Start guiding iterative learning.

A powerful web application for modern educators that transforms Google Slides into dynamic, auto-grading assignments. It provides students with instant, personalized feedback while giving teachers real-time data and unparalleled control.

Tech Stack: React, Node.js, MongoDB, Google Slides API, JavaScript

For Students: A Smarter Way to Learn

  • Instant Feedback: Custom grading provides scores and hints, encouraging revision.
  • Personalized Guidance: Unique toast notifications offer tailored support.
  • Persistent Progress: All work is automatically saved.
  • Clear Summaries: A comprehensive summary screen for easy review.

For Teachers: A Complete Command Center

  • Effortless Creation: Paste a Google Slides link to create a new lesson.
  • Powerful Code Editor: Write custom JavaScript grading functions.
  • Real-Time Dashboard: Watch student progress unfold live.
  • Granular Control: Manage assignments, deadlines, penalties, and rosters.

About Us

innovātED was founded on the principle that technology should empower educators, not complicate their work. Our journey began with a desire to create efficient, intuitive solutions for the modern classroom, which led us to build our flagship project, IterātED.

We believe in building robust applications that empower users, whether they are students in a classroom or professionals in a specialized field. We enjoy the entire process of development, from brainstorming and architecture to deployment and iteration, always with the end-user's experience as our top priority.

Here are a few technologies we use to build our solutions:

  • JavaScript (ES6+)
  • React
  • Node.js
  • MongoDB
  • HTML5 & CSS3
  • Firebase
